# PPQQ Technologies Enterprise IT Lab Roadmap

## Project Overview

The PPQQ Technologies Enterprise IT Lab is a long-term hands-on project designed to simulate the IT infrastructure of a mid-sized organization.

The purpose of this project is to gain practical experience with enterprise technologies commonly used by IT Support Specialists, Help Desk Technicians, Desktop Support Technicians, Technology Support Specialists, System Administrators, and Network Administrators.

This lab emphasizes real-world implementation, documentation, troubleshooting, and IT best practices rather than simply following tutorials.

---

# Project Goals

- Build a realistic enterprise IT environment using VMware Workstation.
- Deploy and administer Windows Server and Windows 11.
- Implement Active Directory, DNS, DHCP, and Group Policy.
- Configure enterprise file sharing and printer management.
- Learn Microsoft 365, Entra ID, Intune, and Google Workspace administration.
- Develop troubleshooting skills through realistic help desk scenarios.
- Document every phase professionally in GitHub.
- Build a portfolio that demonstrates practical IT experience to employers.

# Project Phases

## Phase 1 – Enterprise Infrastructure

### Objectives

- Install VMware Workstation
- Create virtual networking
- Deploy Windows Server 2025
- Configure static IP addressing
- Install Active Directory Domain Services
- Configure DNS
- Create the ppqq.local domain
- Join Windows 11 client to the domain

---

## Phase 2 – Active Directory Administration

### Objectives

- Create Organizational Units (OUs)
- Create users
- Create security groups
- Configure user permissions
- Reset passwords
- Unlock accounts
- Disable and enable user accounts

---

## Phase 3 – Group Policy

### Objectives

- Configure password policies
- Deploy desktop wallpaper
- Disable USB storage
- Disable Control Panel
- Map network drives
- Deploy printers

---

## Phase 4 – File Server

### Objectives

- Configure shared folders
- Configure NTFS permissions
- Configure share permissions
- Create department file shares
- Map network drives

---

## Phase 5 – Print Server

### Objectives

- Install Print Services
- Deploy shared printers
- Configure printer permissions
- Deploy printers using Group Policy

---

## Phase 6 – Microsoft 365

### Objectives

- Create Microsoft 365 tenant
- Configure Entra ID
- Create users
- Assign licenses
- Configure Microsoft Teams
- Configure OneDrive
- Learn Intune fundamentals

---

## Phase 7 – Google Workspace

### Objectives

- Create users
- Configure Organizational Units
- Create groups
- Configure Shared Drives
- Configure Chrome policies

---

## Phase 8 – PowerShell Administration

### Objectives

Learn practical PowerShell administration including:

- Get-ADUser

- Set-ADUser
- Unlock-ADAccount
- Get-Service
- Get-Process
- Get-ComputerInfo
- Test-NetConnection

## Phase 9 – Help Desk Simulation

### Objectives

Create and resolve realistic support tickets including:

- Password resets
- Printer issues
- DNS problems
- DHCP issues
- Network drive failures
- Shared folder permission issues
- Outlook problems
- Windows login issues
- Wi-Fi troubleshooting
