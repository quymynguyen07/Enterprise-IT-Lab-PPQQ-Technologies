# PPQQ Technologies Enterprise IT Lab

## Overview

Welcome to the PPQQ Technologies Enterprise IT Lab.

This project simulates the IT infrastructure of a fictional mid-sized technology company. The purpose of this lab is to gain hands-on experience with enterprise technologies commonly used by Help Desk, Desktop Support, System Administrator, and Technology Support Specialist roles.
